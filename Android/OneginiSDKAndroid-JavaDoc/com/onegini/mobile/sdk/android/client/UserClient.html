Class UserClient

  • All Implemented Interfaces:
    
public final class UserClient

    User client implementation.

    Main responsibilities:<br></br>

    • handles users authentication<br></br>

    • logs out users<br></br>

    • deregisters users<br></br>

    • registering authenticators <br></br>

    • provides Retrofit Client instance for resource calls<br></br>

    • provides access token for the currently authenticated user<br></br>

         final Unit authenticateUser(UserProfile userProfile, OneginiAuthenticationHandler authenticationHandler)

        Authenticates a specific profile. The authenticated profile will be returned with a success callback.

        This flow checks, if (encrypted) refresh token is available, then uses the preferred authenticator to decrypt the refresh token in order to obtain an access token.

        Parameters:
        userProfile - the UserProfile that we want to authenticate, the UserProfile object can't be null
        authenticationHandler - the authentication handler to return the authentication result to

      • authenticateUser

        Authenticates a specific profile using a specific authenticator. The authenticated profile will be returned with a success callback.

        This flow checks, if (encrypted) refresh token is available, then uses the specified authenticator to decrypt the refresh token in order to obtain an access token.
