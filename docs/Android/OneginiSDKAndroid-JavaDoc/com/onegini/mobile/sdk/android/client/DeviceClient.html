Package com.onegini.mobile.sdk.android.client

Class DeviceClient

  • All Implemented Interfaces:
    
public final class DeviceClient

    Network client implementation.

    Main responsibilities:<br></br>

    • Authenticating the device.<br></br>

    • Providing OkHttp clients secured by the SDK for fetching (authenticated) resources. Authentication is only based on the device (client) credentials no user interaction is required in order to authenticate.<br></br>

      • authenticateDevice

         final Unit authenticateDevice(Array<String> scopes, OneginiDeviceAuthenticationHandler deviceAuthenticationHandler)

        Performs device authentication. Uses the device (client) credentials to request an access token object, which can be used for performing anonymous resource calls. We also sometimes refer to a device as a client, in that case we refer to the concept of an OAuth client.
